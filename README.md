# Project_EOR
Supervised ML Codes (Adaboost, Random Forest, Ridge Regression, SVR, and XGBoost) for the EOR Paper
**https://doi.org/10.1039/D4CP04724A**

A Sample .csv file is added (the same was used in the paper as well, and corresponds to ML-1 group of dodecane oil model (Model-1))
Python scripts for feature importance selection (SHAP Score), and the rest of the Supervised ML codes are added.
